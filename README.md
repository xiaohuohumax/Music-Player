# Music-Player

**简洁音乐HTML播放器**

## 播放器特点

+ **播放器可以随意拖动**
+ **自动根据网页大小变换大小和位置**
+ **简单易用**
+ **全新界面设计**
+ **基于less ,Vue ,jQuery 修改简单**
